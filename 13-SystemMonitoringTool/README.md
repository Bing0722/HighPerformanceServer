## 系统监测工具

### tcpdump
tcpdump 是一个强大的命令行网络抓包工具，可以捕获网络上的数据包并进行分析。

### lsof
lsof（List Open Files）是一个列出当前系统打开文件的工具，网络连接和文件系统上的文件都可以通过该工具查看。

### nc
nc（netcat）是一个功能强大的网络工具，可以用来读写网络连接，进行端口扫描、调试等。

### strace
strace 用于跟踪进程执行过程中系统调用的情况，能够帮助调试和分析程序执行的底层行为。

### netstat
netstat 是一个用于显示网络连接、路由表和接口统计信息的工具。

### vmstat
vmstat 用于报告虚拟内存、进程、CPU 活动和其他系统资源的统计信息。

### ifstat
ifstat 用于显示网络接口的实时流量统计。

### mpstat
mpstat 用于显示各个 CPU 核心的使用情况。
